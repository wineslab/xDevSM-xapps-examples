# Radio Bearer xApp

Example of Radio Resource Bearer Control RC xapp using the xDevSM framework.


This xApp sends a Control Request containing a `QoS flow mapping configuration`.

### Options
```python
  -d <drb_id>, --drb_id <drb_id>
                        DRB ID
  -q <qos_flow_id>, --qos_flow_id <qos_flow_id>
                        QoS Flow ID
  -m <qos_flow_mapping_indication>, --qos_flow_mapping_indication <qos_flow_mapping_indication>
                        QoS Flow Mapping Indication
```
